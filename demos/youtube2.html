Able Player for YouTube (Complex Example)

This page features a relatively complex example of Able Player being used to play a YouTube video. Its features include:

  • A described and non-described version of the video, both hosted on YouTube.
  • Captions, subtitles, and chapters; all hosted locally.
  • An interactive transcript, only possible since the above timed text resources are stored locally.
  • An external container to which Able Player adds chapters.

For additional demos see the Index of Able Player Examples. In particular, check out the Simple YouTube Examples page.
