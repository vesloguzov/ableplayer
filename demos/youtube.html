Able Player for YouTube (Simple Examples)

This page features a few simple examples of Able Player being used to play YouTube videos. In all examples, the video is available with and without audio description; both versions are stored on YouTube, and can be swapped using the Description button.

For additional demos see the Index of Able Player Examples. In particular, check out the Complex YouTube Example.

Video #1

All assets, including captions and poster, are on YouTube. Width & height are defined by CSS of a parent element. The video is available with and without audio description (both versions are stored on YouTube, and can be swapped using the Description button.

Video #2

This is the same as Video #1, except for how width and height are defined. Both are defined using the width and height attributes on the video element.

Video #3

Captions and poster are stored locally, not on YouTube. Using local captions makes the automated transcript possible. Width and height are the same as Video #2.
